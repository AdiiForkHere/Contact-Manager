# 📇 Contact Manager (Flask)

A simple and efficient **Contact Manager Web Application** built using **Flask (Python)** along with **HTML, CSS, and JavaScript**. This app allows users to manage contacts by performing CRUD operations (Create, Read, Update, Delete).

---

## 🚀 Features

- ➕ Add New Contacts  
- 📋 View All Contacts  
- ✏️ Edit Contact Details  
- ❌ Delete Contacts  
- 🔍 Search Contacts (optional)  
- 💾 Data Storage (SQLite / JSON / In-Memory)  
- 🎨 Clean and Responsive UI  

---

## 🛠️ Technologies Used

- **Python (Flask)** – Backend framework  
- **HTML5** – Structure of pages  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Interactivity  
- **SQLite** (optional) – Database
